※ 引述《reader (讀者)》之銘言： : ※ 引述《wnlee (Taiwan 1)》之銘言： : : 為甚麼記者會發明出台灣十大電腦高手這個名詞呢? : : 有哪位先進可以介紹一下十大電腦高手的故事? : 《我的電腦探索》是《資訊游俠列傳》的姐妹作，作者為：黃志賢、全柏樺、 : 劉燈、葉秉哲、單中杰、戴凱序、侯俊傑、唐宗漢、賴明宗、黃景彤。這算是 : 我主編的書當中最花心思的一本了。 黃志賢、劉燈、葉秉哲、單中杰、賀元、唐宗漢、賴明宗這七個都算是有一些 名氣的（原本的名單是賀元而不是侯俊傑），故事很多很長又有很多人知道， 就不多說。 全柏樺是應用派的，作為高中生雖然算是強人，但故事相對起來比較平淡也就 不提了。 戴凱序主要是單中杰找來合寫文章的，不太算是主要名單，他個人雖然聰明， 但應算是化學天才而不是電腦天才，電腦方面的故事則主要集中在黑客文化的 提倡方面，其他像是寫出 BASIC compiler 或 GUI system 都偏向遊戲之作， 算不得準（當然就高中生而言算滿強的了），這裡也就不多提了。 特別拿出來講的是黃景彤，當年他唸國二，和唐宗漢同年，算是當時最著名的 兩位國中生（也是建中資訊社空前絕後的四位國中生社員其中二位）。在電腦 方面的能力集中在網路上，那時就是幾家網路公司的顧問，對於網路技術算是 熟練，著名事蹟是 "不小心" 就進了台北市政府的網站主機取得 root. 以國中生而言，黃景彤當時比唐宗漢更有名，本來國中生只有黃景彤一個，而 唐宗漢是自己跑來說要參加寫作計劃才塞進去的。 黃志賢和葉秉哲是到了大學才出名，此外都是中學就出名了，劉燈更是國小時 就作出著名軟體，所以這本書也才常常被一些老軟體工程師嫌，說什麼看了會 覺得沮喪之類的。 其實除了我們這些人之外，也想過去找台大或交大的學長姐，兩個學校都可以 找出一整組的強者，再往上找現役工程師也可弄出一組高手出來，矽谷的華人 又還有一堆強人，美國各大學裡頭也有一堆，但一來年齡不同、環境不同，所 關注的東西就不同，找他們出書的主題也會不一樣，而這系列賣得不如預期， 比起出 Word 手冊幾千本幾千本的增印完全不能比較，結果就沒有繼續做了。 也因此，除了《資訊游俠列傳》和《我的電腦探索》之外，就再也沒有整份的 名人名單了。當然許多雜誌都有名人採訪，但是比較有人味兒的企劃，則沒有 其他的了。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 61.222.173.29
