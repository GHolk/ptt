標題Re: [討論] 為何要將條目刪除?
時間Tue Dec 7 09:35:38 2010
※ 引述《SmallBeeWayn (喵喵叫的蜜蜂貓)》之銘言： : 本人不算是維基人，實際參與過編輯的條目也不過就兩三條而已 : 編輯最多次的反而是自己的User頁.... : 我比較接近的是程式與網站設計師 : 所以，請讓我從系統的角度去切入「刪除條目」這個議題 : 為甚麼條目必須被刪除? : 1.降低系統負載 : 簡而言之，這是詭論... : 對於一般索引方法的網站而言，10萬筆跟20萬筆資料是兩倍搜尋時間 : 如果RAM不夠大，可能不只兩倍時間 : 但對於使用複雜演算機制進行索引管理的網站而言(我相信維基是) : 在10萬筆跟100萬筆中搜索所需要的時間差異，小於網路延遲時間 : 至於儲存空間部分，就算是最昂貴的SLC SSD也只要1GB/NTD.100 : 一篇鉅細靡遺的條目，應該不會超過50KB(經驗估計，我沒有去查驗) : 而一篇內容不足的條目可能只有5KB : 為了節省不到NTD.0.0005的儲存成本而刪除條目，顯然沒有意義 : 當然，除了本文以外，索引也會占用儲存空間 : 但不管怎麼樣去計算 : 儲存與索引引發的額外成本都很難達到影響網站維護成本的規模 : 網路流量的部分.... : 既然這個條目沒有受到關注，就代表不會有網路流量消耗在這個條目上面... 我覺得這應該不是重點，雖然能省的應該省，不過因為省而刪不贊成。 : 2.關注度不足 : 從系統管理的觀點而言: 沒有差別 : 從使用者的角度而言: 反正用不到 : 從建立(條目)者的看法而言: 我覺得有建立的價值 : 沒有人知道這個默默無名的條目未來會不會廣受注目 : 只要內容正確，沒有侵權或其他問題，把它擺在那裏也沒有礙到任何人 : (提刪者)為何要刪除? 其實目前中文維基所謂「關注度不足」 不是「知名度不足」或「默默無名」 的問題。 還是有點不同。 關注度不足是指沒能獲得第三方有效介紹的主題。 換言之，除了撰寫者本身（或該事物一方本身）以外，找不到其他 可供查證 的可靠來源。 也就是說，從其他編者及讀者的角度來看，我們沒辦法確定它的內容 是真是假、是中立還是其實是宣傳成分。也無從幫忙作出較好的改進。 另一個相關的就是「宣傳」 維基百科不應該作為一個公司團體宣傳的網站。 維基條目的 pagerank 十分高， 如果維基任何小網站、小公司都照登不誤， 如果我是網站或公司的人，當然不會放過這個免費廣告的機會。 這也是為什麼遇到團體、企業、網站的條目，關注度又會特別被重視。 （其他如書籍、專輯等等也很類似） 至於「未來可能會有關注度」「創建者認為有價值」 講得極端一點，假如一個人說我家的小baby未來可能會當總統，所以我建立他 的條目介紹他的資料生平，你覺得維基百科應該要收錄嗎？ 的確是大多情況下 不礙其他人沒錯。但維基百科應該拿來作這種事嗎？也許每個人會有不同的看法就是。 我贊同關注度不應該過於嚴格，對於一些可能有關注度的主題應該是儘量找尋來源 而不是尋求刪，那樣會搞到水清無魚。但關注度這個要求我覺得基本上是合理的。 : 3.小小作品 : 基本上就是1&2的合併 : 這些條目並不會造成任何妨礙，而且有助於(交互參照)結構的完整 基本上我是贊同小小作品（主題有明確定義，有少許內容）有保留的價值。 不過無法改變認為該刪的維基人也沒辦法。 : ============= : 那到底甚麼該刪了? 我的看法大概就只有這種 : 「嚴重謬誤(到整篇重寫比較快的程度)，杜撰的條目，主觀意見」 : 作為一個百科，我認為最重要的就是客觀、事實描述與中立性 : 重複的條目可以合併 : 不照標準走的條目可以改正 : 缺乏來源的條目就補足或是掛上模板 : (只要描述足夠客觀(大眾化)，畢竟不是所有資料都還找的到來源) : 辭典的條目，有維基字典可以落腳 : 自傳? 請客觀中立，並描述事實 : ============ : 其實我覺得就看這篇 http://goo.gl/jZkVX : 只要沒有違反幾個負面表列的項目的條目，都沒有刪除的必要 : 倒是這篇 http://goo.gl/f7mxh 我看不懂... : 意思是說不能虛構一個條目，但如果它是個現實世界存在的作品(或是類似的東西) : 就可以這樣嗎? fuzuki有很好的說明了，主要是指 作品中的事物而言。總之就是 太不重要的虛 構角色事物不能有獨立條目。 ---------------- 整體而言，我是贊同現在在執行面上，中文維基百科對於刪除 過於嚴格以及過於「隨手」，而刪掉了很多應可保留的東西。 但以制度來說，現有的制度（例如小小作品的刪除規則、關注度的刪除規則） 其實就是參與維基百科各方意見妥協找到的公約數 （例如在以前小小作品或關注度不足的內容，可能是直接提刪沒有緩衝時間 現有制度也是保留派一方爭霰的結果） 這些不是說不能進一步去改變，而是目前要取得社群共識而改變並不容易。 當然願意嘗試改變大家的想法也是很好的，但「改變」一樣是要「去實作」 ，去說服其他人改變對於維基百科的基本看法，而不是只是說自己的看法道理， 就能輕易改變大家的想法。 在共同創作的特性下，如何找到不同看法之中 的公約數，才能解決問題。 在現有的制度下，去爭取在執行上的合理性我覺得是比較可行的。 Alberth兄 就做了非常多，真正是救了非常多條目， 我自認近來這方面不夠努力，就只有 偶爾去投一些保留票，或補充幾個小小作品。 ------ 進一步我覺得有些想法的差異，牽涉到每個人對於維基百科的定位想法， 也就是維基百科是否有需要「因為無礙於其他人」就什麼都收錄。 我覺得中文世界很多人把維基百科當成這種什麼都收錄的角色，不過像在英文世界 （德語、日語、西語…也是）在wikipedia之外，都有一大票很活躍的wiki站，收錄 各種不同主題的，更細鎖更小的事物，這種情況在中文世界裡比較少（目前比較成 熟活躍的站大概就是 棒球維基館、香港網絡大典、香港巴士大典 等等） ， 大家也還比較不熟悉這種主題站的形式，所以變成什麼都覺得應該放在維 基百科放，我覺得其實並不是很必要。其實可以讓不同的wiki站發輝不同的效果。 例如，作品的小角色到作品的wiki站、明星fans的內容到明星的wiki站、很小的地名到地區 的wiki站，小網站到網站的wiki、小公司到類似黃頁的wiki。 另外很多語言其實都有成立那種「收錄wikipedia不要條目」的wiki站。 中文其實也可以來成立一個，畢竟很多被刪的內容真的很可惜。 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 203.70.51.82 ※ 編輯: ffaarr 來自: 203.70.51.82 (12/07 09:36)
