我不是品酒高手 但是我喝過這支"96%的生命之水" 來說說感想 也讓未來想嘗試的大大有個評定的基準 首先"一定要凍飲"不凍飲我想應該要馬上送醫院吧 我喝過的喝法 1. +啤酒 就是版上說過的"超級深海炸彈" 我的比例是 50cc生命+330cc金牌台啤 意外的好喝 高酒精的生命 完全把低酒精的台啤的甜味提出來 強烈推薦試看看 但是 一加其他種的啤酒就悲劇了 難喝到有苦味 ( 我加過7-11自有品牌 ) 2. 純飲 ( 前面有說一定要凍飲 ) 因為我跟同事都很怕死 所以我們冰了48小時 然後倒出30cc一口喝光 ( 不用冰杯可以喝啦 我們人體實驗過了 ) 不誇張 一點感覺都沒有 但是10秒後 喉嚨就有酒精脫水的感覺 ( 超級強烈 ) 馬上喝水也沒用 大概持續10~20分鐘 ( 看每個人的體質 ) 那種感覺很奇妙 我也說不上來那種脫水的感覺 痛痛的 爽爽的 真的算是不舒服 但是又很奇妙 通常我們走完上面2個run 再聊10分鐘( 包含收酒杯 ) 就收工各自回家 總時間約30分鐘吧 對我們來說 喝生命就是要省時間 走路回到家 洗澡洗一洗 效果剛好出來 立馬睡覺!!! 96%生命之水說真的居家必備 便宜 效果好 又不花你太多時間 只是需要一點點勇氣 ※ 引述《xiangming (鄉民)》之銘言： : 聽聞這隻生命的水很久了 : 但對這種賣弄酒精的產品一直興趣缺缺 : 最近買酒時無意間看到 : 抱持著好奇的想法順便帶了瓶試試 : 當然這種東西拿來調酒就沒什麼特別的了 : 當然要試試酒精的味道 : 一開始怕受不了就先加了點冰塊試試 : 喝起來順口順口的 : 就冰的伏特加但高酒精度的關係 : 還是免不了食道脫水帶來的刺激感 : 所以加冰並不特別 : 又倒了點純飲試試 : 也稱得上順口 : 味道並不會太強烈但明顯感受的到酒很"純" : 也沒有想像中在口中瞬間揮發的快感 : 但畢竟是純飲脫水感帶來的乾澀更為強烈 : 也無法再多喝 : 純飲不推 : 再加點水來試試 : 2:1 酒 水 : 很難喝 : 就變成一般伏特加了 : 就是酒精加水 : 配吃的也不好喝 : 最後想說應該是拿來調酒比較適合 : 無奈身邊又沒什麼東西可以調 : 就拿冰箱的台啤來炸 : 不錯真的是拿來調酒的 : 平常的烈酒拿來炸酒精味很濃 : 這隻反而很融合 : 勉強可接受 : 結論 : 喝了1/3瓶心得是 : 不知道怎麼喝這隻酒 : 就只是隻高酒精度的伏特加 : 後力很強走路一直被看不到的力量踢 : 價格覺得不值得 : 現在丟在雪櫃裡凍 : 有熱心的各位知道這隻要怎麼喝的嗎 -- ※ 發信站: 批踢踢實業坊(ptt.cc) ◆ From: 125.230.56.42
