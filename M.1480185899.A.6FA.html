標題Re: [新聞] 卡斯楚90載傳奇一生 躲過634次暗殺
時間Sun Nov 27 02:44:57 2016
古巴目前只有早上新聞有報以外，在街上或家裡沒什麼人在討論這件事，當然古巴的里長 組織CDR（這組織就跟台灣里長很像，我岳母還是她們社區的長官）也會收集輿情然後報 告上去，當地還是有很多事情是不能公開講的。 卡斯楚是古巴革命三傑之一，除了大家比較熟悉的切格瓦拉以外還有Cienfuego Camilo， 分別有他們各自的紀念日，卡斯楚是生日；切是逝世紀念日；Cienfuego則是失蹤紀念日 。沒錯就是失蹤紀念日，當初他是古巴第二把交椅，但某天政府就宣告他失蹤了。卡斯楚 則是穩定執政近半世紀。 古巴人在古巴經濟危機時最懷念的就是Cienfuego，大部分的人也有共識認為Cienfuego 就是被卡斯楚暗殺，但這也是禁止討論的事，只是臆測但又不能說明。古巴最好玩的就是 曾經的革命戰士最後都成為能把他們名字刻上去的各種紀念碑或銅像。 這些著名戰士卻只有卡斯楚一家族存活下來。不要再說卡斯楚反對個人崇拜，古巴的小學 要唸他們家的家譜，還要每間小學製作一本結合相片跟寫給領袖的話的小冊，在生日那天 要獻給他，然後他九十歲生日聯播一個月以上的電視宣傳，都是各行各業各年齡層的獻花 啦什麼。 但古巴人到底喜不喜歡卡斯楚，我想因為革命受惠的就會很喜歡，鄉下資訊封閉不與外界 接觸的古巴人更是。我太太她們家是地方望族，古巴革命沒收所有產業，直接從工廠老闆 變工廠領班，聽她外公講他爸爸還因為反徵收被政府槍斃。 原本幫他們家工作的人一天之後有了自己的房子跟政府分配的工作生活確實得到改善。但 我太太她們家族就一半以上移民到國外，留在古巴的親友只能透過移民的親戚接濟。這些 人就不用想極度痛恨古巴政府。 古巴一到現在仍然不會跟人民收稅，只有企業或是有盈利登記的個體戶才要繳稅，房地產 、車子也是直接買斷制不用交任何的稅，但如果無人繼承會收歸國有。在不用繳稅的情況 下，搭配堪用的免費醫療（可翻我文章）和教育（學校還有免費營養午餐喔）以及固定的 食物卷配給，還是有很多古巴人民很多熱愛政府的。 但見過世面接觸過觀光客的那些古巴人就超級討厭政府，但不熟的時候當然還是會跟你說 古巴有多好多好就是了，但喝了幾杯還是會找你聊政治罵政府跟吹牛。 ※ 引述《ru04ul4 (拒絕)》之銘言： : 卡斯楚90載傳奇一生 躲過634次暗殺 : 2016-11-26 21:39〔即時新聞／綜合報導〕 : 古巴前領導人卡斯楚（Fidel Castro） : 25日深夜逝世，享耆壽90歲。 : 一代強人離世震驚全球，也留下許多驚人紀錄， : 包括其是世界在位第3長的國家領袖、 : 曾遭到暗殺高達634次， : 還創下聯合國演說時間最長紀錄。 : 《路透》整理出數個卡斯楚特別記錄： : 1.在位任期第3長的國家領袖。 : 卡斯楚領導古巴長達50年， : 擔任國家領袖時間僅次於 : 英國女王伊莉莎白二世及 : 今年10月逝世泰國國王蒲美蓬。 : 卡斯楚在2006年7月接受腸道手術後， : 暫時將權力交給弟弟勞爾卡斯楚（Raul Castro）， : 2008年正式交棒。 : 2.創下聯合國最長演說時間。 : 根據聯合國網站，卡斯楚曾於1960年9月26日， : 在聯合國進行長達4小時29分鐘， : 寫下最長紀錄。而卡斯楚本人最長演說紀錄 : 則是在1998年2月24日，古巴國民議會再次 : 選他出任總統後，卡斯楚發表長達7小時演說。 : 3.遭暗殺達634次。 : 卡斯楚宣稱，他逃過了634次暗殺， : 其中要是由美國中央情報局（CIA） : 及美國古巴流亡組織所策畫， : 暗殺手段包括毒藥丸、毒雪茄、爆裂物。 : 其中一項是利用粉末讓卡斯楚的 : 招牌鬍子掉光，破壞他的人氣。 : 4.經歷美國9位總統。 : 卡斯楚領導古巴50年， : 期間面臨美國經濟制裁及豬灣事件， : 但仍然屹立不搖，撂倒9個美國總統， : 從艾森豪（Dwight Eisenhower） : 到小布希（George W.
