標題[新聞] 爆炸原因找到了！傳三星本月份將公開說明
時間Mon Jan 2 13:40:42 2017
1.媒體來源: 自由 2.完整新聞標題: 爆炸原因找到了！傳三星本月份將公開說明 3.完整新聞內文: 三星 2016 年下半年經歷了有史以來最嚴重的危機─旗艦手機 Galaxy Note 7 爆炸、召 回、停產。不過至目前為止，三星都尚未公開對外宣布 Note 7 爆炸的真正原因。不過今 天《路透社》報導指出，三星將會最快於本月就公開 Note 7 起火燃燒的正式調查結果。 Galaxy Note 7 於今年 9 月初首度傳出起火燃燒事件，當時三星初步調查 Galaxy Note 7 由 SDI 提供的電池有隆起的現象，但另外一家廠商提供的電池卻沒有，雖然當時還沒 有查清楚為何電池會隆起，但因為來自消費者、通路、電信商的抱怨不斷湧出，三星行動 通訊事業部總裁高東真決定「當機立斷」，下令召回 250 萬支 Galaxy Note 7。 證據尚未搜集完整的情況下宣布召回，是 Note 7 斷魂的第一步。兩個星期後，三星送出 數百萬支更換新電池的 Galaxy Note 7，結果仍然發生多起爆炸事件，證明了電池根本不 是主要原因。在無法找出起火原因的情況下，三星決定直接將 Note 7 一槍斃命，宣布永 久停產。導致 Galaxy Note 7 成為三星史上最短命的智慧手機，也讓三星損失超過 50 億美元，更賠上了苦心經營許久的招牌與消費者信心。 其實上月月中，就有外媒指出三星其實已經將 Note 7 爆炸的原因找出來了，且已經向國 際知名認證機構 UL（Underwriters Laboratories）和 Korea Testing Laboratory 分享 內部調查結果。不過三星目前仍處於高度保密狀態，也尚未對外宣布確切公布 Note 7 爆 炸的時間與細節。 時序進入 2017 年，三星的全新旗艦手機 Galaxy S8 預計在不久的將來推出。為了要挽 救下滑的消費者信心與商譽，三星應會選在新手機發表之前，針對 Note 7 爆炸事件給全 球消費者一個完整的交代。不然消費者心中疑慮未除，新手機的銷量難免將勢必受到影響 。 4.完整新聞連結 (或短網址): http://3c.ltn.com.tw/news/28093 5.備註: -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 140.115.204.99 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1483335644.A.753.html
