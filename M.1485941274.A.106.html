身為一個修練多年的職業魔法師 我想我有必要出來導正視聽一下(蛤? 物理(physical)是物質的理論 而魔理(magical)便是魔法的理論 就如同 物理有四大力學(超~崩~潰!! 魔理也有四大屬性 地水火風 首先要先知道 魔理理論與物理理論之間最大的區別是 「魔理可以輕易的實現"逆做功" 而物理則不行」 除此之外 實際上物理與魔理在理論之間是息息相關的 - 首先是[地魔法] 一言以蔽之 地魔法就是可以改變"物質特徵"的屬性 不管是應用於 煉金術 物質強化 戰鬥中的土之槍 還是要刻一座美女像 他的原理是透過操作{分子間作用力}(電磁力) 來實現物質的變形 甚至是任意氧化還原 也就是 地魔法其實就是操作電的魔法 但這只是其中一種性質表現而已 不管是口語上的 直流電`靜電還是分子力 全都是電磁力的表現 其中 分子力又包含{化學鍵`氫鍵`凡得瓦力}巴拉巴拉一大串的 太囉嗦了跳過不說明 但畢竟地魔法也就只是操作電磁力的程度而已 所以雖然萬用 但依然必須遵守質量守恆的定律 絕不可能無中生有創造出質量 但因為電磁力理論的實用性與快速發展 所以地魔法的魔法革命開啟了文明技術的大躍進 是帶動社會進步的重要魔法 至於某科學方的逼哩逼哩 號稱是"最強電磁系能力者" 但居然卻連最基本的氧化還原術都做不到 令人強烈的質疑 該人是否只是仗著年輕的肉體出來賣弄話題的網紅神棍而已 這種糜爛的風氣實在是非常不可取 要特別注意的是 在操作不同的元素時 所耗費的能量是差異非常多的 關於操作物質時所需要耗費的能量 請參考{標準電極電位}表 - 接著是 [水魔法] 水魔法是利用古典力學來操作質量運動做功的一種單純而樸實的魔法 之所以用水做為代表 是因為液體是最難操作的複雜物體 需要用複雜的數學運算式 才能正確的表現出流體的連貫運動 所以 凡是能夠流暢操作液體的水魔法使 必然能操作同等以上的固體 而隔空取書等等的基本技巧 雖然是幼稚園程度就會教的最基本入門課程 但是 複雜一點的像是水槍術 甚至是高階水龍術 就必須以非常冗長的數學式來表現其運動 同時在施放時必須有很長的前置詠唱準備時間 為其使用上不利的一面 - 然後是最熱門的[火魔法] 火魔法是最受歡迎的屬性 同時也是最簡單的一種魔法 其本質就只是很單純的能量施放 然後轉換熱量做功而已 連基本的熱力學都稱不上 但就因為其原理簡單 所以易學易上手 不須太多的思考就能使用 應用在戰場上也很受歡迎 所以成為最熱門的魔法屬性 但是要注意的一點是 火魔法其實是一種典型的富豪魔法 所謂的富豪魔法就是 施展時的魔力耗費巨大 若非天生的魔力天才則難以長時間使用 舉例來說 水魔法中的普通水槍術 僅需500焦耳(9mm子彈484J)不到的能量 便可有效至人於死地 而同樣最基本的火球術 若要讓一個70公斤的成人 加熱到立即至死程度(大約80度) 卻必須耗能 70kg * 3500 J/kg℃ (人體比熱) * 40℃ ≒ 9.8M焦耳 兩者能量消耗差距高達2萬倍以上 所以 當火魔法對上水魔法時 在持久性方面會有壓倒性的劣勢 因此切記務必要避免與其正面對抗 - 最後是[風魔法] 之所以稱為風魔法其實只是種象徵而已 實際上 風魔法是種操作"場"與"波"的環境魔法 透過"力場"的操作產生出壓力 便可自然的起風 甚至是高等重力術 可對大範圍進行無差別壓潰 即便是以堅固著稱的地魔法 面對波的攻擊也難以抵抗 是種非常擅長面對大範圍的屬性 - 以上就是四大基本屬性的介紹 另外 關於標題問到的冰魔法 冰魔法是種高等魔法屬性 雖然兩者可以共用物理介質 但實際上冰魔法與水魔法是完全不一樣的概念 雖然常見的冰槍術使用了水做為施展介質 並予以操作 但在冰結術中並不需要介質便可直接冷凍而牽制對象 前面提過 水魔法只是純粹的古典力學 無關乎性質的變換 但是冰魔法卻是 顛覆熱力學第二定律的異質魔法 透過把熱力學的逆轉 實現了熵的逆流 從常溫中分離出冷熱 所以 冰魔法確實是不等同於水魔法 是種非常艱難的屬性 - 最後 附加說明每個魔法使的理想殿堂 "光魔法" 與 "闇魔法" "光魔法" 是創造的魔法 透過{強力}的操作 光魔法在{核聚變/分裂}上實現了一般上不可能的"無中生有" 以及其龐大的能量輸出 除了透過強力來創造質量以外 若想要創造靈魂時 則必須額外透過%%%的特殊儀式才能完成 所以 某矮豆兄弟當初之所以鍊成媽媽會失敗 一定是事前忘了先去找溫莉做%%%的儀式之故(咳咳咳! - "闇魔法" 能夠消滅一切 最神秘的魔法 實際上 {絕對黑體}便是闇魔法的物質結晶化 黑體是最絕對而不可被探知的黑暗 從來沒有人能夠看到黑體的真正樣貌 只知道在那個地方有一團神秘的純粹黑暗 且黑體能夠吸收任何一切的能量 絕不反射出任何東西的神秘物質 因此 任何形式的攻擊在黑體面前都是毫無意義的的 只會遭受到無盡的吞噬 另外 闇魔法最常見的應用便是知名的"鐵壁短裙" 又人稱薛丁格的內褲 其強大的力量 令人無論在在強風或是倒立等等任何環境之下 都讓人無法探知到裙底之下的任何風光 這種能夠顛覆一切物理力學的現象 堪稱是最強大的防護魔法也絕不為過 而非常特別的是 幾乎所有的表番女性角色 都會天生的自動內建短裙鐵壁天賦 就連貴為神族(雖然是個智障)的NO胖女神 都同樣會使用這個闇魔法 (而神族居然使用闇魔法又是另一件令人爭議的後話了) 但是只要到了里番之後 該天賦反而會遭侵蝕而必然消失 此為魔理學的七大不思議之一 - 施展魔法時必須消耗對應的魔力 而魔法最令人感到不可思議的地方就是 魔法其強大的力量來源 "魔力場"(部分流派稱之為氣場) 過去 魔理學界一直無法正確解釋魔力場的能量來源 但在魔理學長年不懈的研究之下 終於發現了"魔磁感應定律" 該理論完美的解釋了這個長年高掛的謎題 過去在法拉第定律中 發現了電場與磁場交換的規律 且磁場力量遠大於電場 而在魔磁感應定律中 則建構了魔力場與磁場交換的關係式 且魔力場同樣遠大於磁場 因此 在經過了2次的能量增幅轉換後 讓魔力場發揮出驚人無比的力量 這也是魔法之所以偉大 最令人著迷的地方 - 至此「簡單易懂的基礎魔法學講座」 全部到此結束 有任何不懂的問題 歡迎發問 -- 我想 我真的是太閒了(=_=a ※ 引述《bear26 (熊二六)》之銘言： : 用簡單的魔法原理來說 : 法師 薩滿等可以操控瑪娜 : 一個是把大氣中的水元素萃取出來 : 凝聚 增幅 然後放射 : 一個是把大氣中的水元素 : 凝聚 降溫 凝結 增幅 然後放射 : 怎麼看都是冰魔法比較高階 : 所以按這個邏輯看 : 基礎魔法應該是地 風 水 暗 光 : 而火 冰屬於進階法術 : 火應該是把大氣中的熱能給凝聚 : 然後增溫 增幅 催化（需要可燃物）點燃 : 然後放射 : 所以火又比冰難一些 : 然後更難是電 : 把大氣中的能量凝聚 : 分解成熱能和冷能 : 增幅 震盪（熱能和冷能摩擦） : 再來放射 : 電要同時可以可以控制高低溫 : 會雷電的基本一定會冰火法術 : 凝聚這件事情是取決於熟練（施法速度） : 還有該地存在的元素量決定 : 增幅這件事取決於施法者本身的能量決定 : （威力） : 真的有魔法 : 最基本課程一定是凝聚 而且是風魔法 : 我認真了 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 118.163.72.102 ※ 文章網址: https://www.ptt.cc/bbs/C_Chat/M.1485941274.A.106.html
