的頁面分類中看到「管理員」的分類。 但有些用戶頁上會有琳瑯滿目的 標籤（右側） 和分類（底部）上面有各種「頭銜」 ，所以很容易就會讓人覺得這個人是管理員。但其實不是。 這些頭銜可分為三大類： 一、榮譽獎勵類：這類頭銜是根據特定的規則給予該維基人的，純粹是榮譽而 沒有任何實質的權限，以ptt來說，有點像是 風雲人物票選上榜的鄉民，但 不會獲得任何管理權限一樣。 如果看到「xx編輯」「xx特殊貢獻」、「xx專家」、「xx寫作獎」的標籤或分類 就是這一類的頭銜。 二、 工作小組、興趣小組或其他組織類： 這類頭銜主要是自己加的，用戶加入特定的工作小組、興趣編輯小組或其他 維基人組成的組織，參與相關工作，就會放上特定的頭銜，也一樣沒有任何的 權限。以ptt來說，有點像是在自己的用戶介紹寫我 「我是XX板板友」一樣。 常見的例如 「風紀委員」「文字工匠」、「格式審查員」、「維基化督導員」 等等。 三、真正與權限有關的是mediawiki系統的各用戶組。 在中文維基百科有用到的用戶組 如下： 1、匿名用戶：未有帳號，以ip顯示 2、登入用戶：有帳號 3、自動確認用戶：註冊7天，50次編輯 上三個是屬於一般用戶的用戶組，但只要成為3之後，幾乎所有一般的功能都能使用， 包括 創建頁面、上傳圖片、編輯條目（包括增加、修改、刪除內容、放置要求改 進模板）、留言（包括討論、建議、投票、提出條目刪除）等。 4、ip封禁例外： 因為有些進行大量破壞的 ip 會被封，怕封到一般用戶而設， 也是屬一般用戶會用到的。 接下來幾個用戶組雖然有一些不同的權限，但主要是作為增加便利性或輔助性的工具 而並非在編輯條目時擁有 一般用戶做不到的權限。 5、巡查員： 和管理員一起擁有標註條目為「已巡查」（也只有巡查員、管理員 會看到）的權限， 但這主要是作為合作檢查新條目輔助工具，一般 用戶一樣能參與新條目的檢查和編輯。 6、巡查豁免： 新創的條目直接視為已巡查，主要是有經驗編者的條目已不用讓巡查 員費心去檢查。 7、回退員： 擁有一鍵回退其他用戶編輯的權限，但實際上人人都能進行回退， 只是沒回退權要多按幾次而已。 8、機器人：為了怕機器人大量編輯洗板，讓其編輯預設不顯示在最近更改列表上。 ------- 所以說，實際上上述幾類帳號有些不同的權限，但並沒有明顯與一般用戶有太大差異。 真正比較有差異的還是 9、管理員、10、行政員。（註） 其中行政員主要只是在管理員之上，擁有 「賦與和取消管理員」（但需根據票選結果） 「更改用戶名」（有人申請的時候）、「加機器人帳號」（有人申請通過時）等等少數日 常的操作。 所以接下來主要就介紹這篇的主要問卦，也就是管理員。 A： 管理員有些什麼一般人不能做的權限呢？最重要的有三個： a：刪除頁面：相較於一般用戶只能刪除頁面的部分，管理員 可以將整個頁面刪除，但需要由一般用戶提出 「快速刪除」 「侵權刪除」、「討論刪除」 後，依據各自的規定以及討論共識之後 才能刪除。 另外管理員也能恢復已被刪除的頁面，但同樣需依照規定 經過通過刪除申覆之後才能恢復。 b：保護頁面： 在一個頁面發生大量破壞，或是編輯戰的時候，會將頁面暫時保護 讓一般用戶無法編輯，但可以在該頁的討論頁先討論頁面應該要怎麼編 輯，等爭議解決再開放，一般用戶發現這種狀況也可以「申請保護」 讓管理員來執行。 c 封禁用戶：針對破壞以及其他嚴重違反規則的用戶封禁。但也都需要依照相關規則， 一般用戶不能封禁別人，但也可以舉報違規用戶讓管理員來封禁。 B：那麼管理員是不是有特權，是不是作威作福？ 實際上有人認為管理員應該叫「操作員」更適當，因為其實只是進行一般 用戶不能做的工作。相關的基本的規定上就是寫 「管理員沒有任何高於其他用戶的特權，唯能實現社群討論所得的共識」 也就是說任何操作都應該要符合社群所製定的方針、指引，以及討論共識來實行， 當然實際上也許不見得都能這麼理想，也會有管理員胡搞，但如前所述， 在維基百科，管理員隨時作的任何事也一樣是透透明明都可以看到，有違反規則 作威作福的，都可以申訴，確認之後會受到社群指責、要求檢討改進，或更嚴 重會遭到罷免。 C 管理員是怎麼來的？ 經過選舉而來，會有基本的編輯數、參與時間 作為參選門檻，可以自己提名或是 由其他人提名。 由於管理員沒有人數限制，所以每個管理員參選都是一個 獨立的選舉，沒有與其他人競選的狀況。 當選的困難之處在於，要當選管理員需要獲得80%上的維基人支持， 也就是說，一票反對票就可以抵掉4票的支持票。以近年的狀況來說， 要當選管理員難度相當高，幾乎除了要編輯貢獻卓越、 參與許多管理工作（如前所述，不是管理員也都能作管理工作） 表現良好，在中文維基百科沒犯過什麼稍大錯誤，也沒有得 罪什麼用戶（得罪一人可能就代表出來幾張反對票就GG了）參選時還 有回到滿出來的各式各樣問題（有的根本亂問），必須都謹慎妥 善回答以免獲得反對票。 在我看來比選八卦板主可能都還困難一些。 報個卦就是我在十多年前當選中文維基百科管理員的時候，雖然規則沒差很多， 那時的風氣幾乎就是類似鼓掌通過（感覺好像某政黨 XD） 因為參與的沒那麼多，常編輯的人彼此之間都很熟悉，因此大家認為你有能力管理 ，可信任不會亂來，就很容易通過成為管理員了。畢竟沒有人數限制，能 多一個人幫忙處理日常事務都是好。 但現在的選法就有點像真的民眾選舉在選「官員」「民意代表」一樣，這種狀況下 可能也讓大家感覺管理員和一般用戶之間有高低之別，而不是管理員只是 負責操作特定功能的一般用戶。 所以，相較於早期，多數的活躍用戶會成為管理員的狀況，現在只有很少比例的活躍 用戶有辦法成為管理員，因此如果在維基百科上遇到有人刪了自己寫的內容， 有較高的機會你遇到的並不是管理員。 另外管理員也有罷免制度，相較於80%支持才能當選，只要有50%贊成罷免就 能將一個管理員免職，不過以往一般都是犯了明顯違規被發現才會罷免成功。 D 管理員是些什麼人呢？是不是魯蛇？ 管理員以地域來說，台灣、香港、中國大陸 數量都差不多，其他海外華人也不少， 各行各業都有，但一般來說比較活躍的大多還是學生（大學生研究生為主， 也有少數中學生，但一般都是做得別好才會當選），因為比較有空能長時間進行管理， 已經工作的除非工作很閒，一般投入時間就會減少一些，或是幾乎不做事， 因為維基管理員沒有人數限制，所以不會因為工作量少就取消（除非自己要辭管理員， 或是超過半年沒登入帳號，怕帳號被人盜用） 所以像中文維基有80個管理員， 但其中有很多都是像我這樣平常很少做管理工作，偶爾才參與的。 整體而言，是不是魯蛇其實很難判斷。也有可能某管理員很活躍是因為家裡很有錢不用 上班，或是工作就錢多事少，都有可能，當然有些是實際認識的，就多少知道一些。 就我知道的印象而言，我覺得還不算太魯。 ------ （註）：另外還有一些特別的用戶組，非由中文維基百科選出 如查核員、監管員、 或是非常設如監督等，就不多介紹。 : : -- : : 交大 eWant 育網開放式線上課程平台 完全免費學習！ :
