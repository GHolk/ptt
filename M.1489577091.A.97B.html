標題[問卦] 有沒有姆咪發P幣程式的八卦
時間Wed Mar 15 19:24:44 2017
小弟平常有遇到喜事發錢的習慣， 可是最近都用 MacBook，之前那個很好用的程式就用不了惹。 有沒有在 MacOS 也能用的發P幣程式八卦呢? ---- 因為好像沒有，我就只好自己寫惹~ 叫做『姆咪姆咪發錢錢』(MumiGiveP) 大家如果有需要的話就拿去用吧！ [ 特點 ] 1. 支援 Windows, MacOS, Linux。 2. 開放原始碼，為採 GPL 授權之自由軟體。 3. 直接貼 PTT 網址就可以抓到推文名單。 4. 發錢時不用停在 PTT 視窗，可以自由做其他事。 預覽圖 http://i.imgur.com/4KWVqHb.png
DEMO https://youtu.be/kCIcbG_cX0U
[ 載點列表 ] - GitHub Releases: https://goo.gl/zIA9cR - Google Drive: https://goo.gl/HOHaot - 官方網站: https://wabilin.github.io/MumiGiveP ** 檔案有分 Windows 和 Mac 版，檔名附帶 _console 是除錯用版本 ** 建議一般使用者下載普通版本即可，如 MumiGiveP_0.4.1_win.zip [ 原始碼 ] 程式都是用 PyInstaller 包的。 如果懂得怎麼用 Python，建議可以直接下載原始碼執行。 - GitHub https://github.com/wabilin/MumiGiveP (有在用 GitHub 的大大們，願意的話可以幫忙按個星星 >////< (也歡迎參與改良開發喔！ [ 使用方法 ] 執行 mumi.exe 或使用 Python 執行原始碼。 [ Q & A ] Q: 為什麼我用 PTT 網址抓取推文名單，卻少了很多人? A: PTT 網頁版不會立即更新推文，需稍等一段時間。 Q: 勾選 [從剪貼簿抓取] 後，我要在哪裡貼上推文內容? A: 程式會自動抓取您已複製的文字內容，不必再按貼上。 Q: 這個程式要我輸入帳號密碼，會不會偷偷盜我帳號? A: 這是該有的疑慮沒錯。 雖本程式之原始碼皆已公開，也有作者發佈前偷偷改加程式碼的疑慮。 您可以選擇相信程式作者人格，或是直接下載原始碼執行。 Q: 我可以把這個程式轉給別人用嗎? A: 這是一個自由軟體。您可以任意的使用、修改、散佈，甚至販賣本程式。 但必須同時附帶原始碼並使用 GPL (GNU通用公共授權條款) 授權。 Q: 我想要 XX 功能可是程式沒有。 A: 程式目前只包含作者自己會使用的功能，您可以在本篇推文許願， 作者會以慵懶的態度選擇性完成。 Q: 我想贊助作者可以嗎? A: 我希望本程式未來是社群共有資產，因此不會在程式內放入任何贊助功能。 如果您喜歡本軟體，可以幫忙在 GitHub 上點個★星星。 Q: 真的完全不接受贊助嗎?
