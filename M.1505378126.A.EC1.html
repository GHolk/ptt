標題Re: [新聞] 出國賣淫！2台女聲稱打工　遭紐海關攔下
時間Thu Sep 14 16:35:22 2017
大家好，我只是個小商人。 今天看見此主題，我就想做個夢。 -------夢境分隔線------- 我先自我介紹一下，我本業是做跨國貿易。 副業是因為興趣，所以我在洛杉磯有開個小茶店，年資約八年(今年邁入第九年)。 所以今天這個新聞出來，我想小弟我可以幫大家分析一下 現在景氣差，這個行業的錢來的快，因此已經有越來越多台灣妹子出國打工了 以前台灣的妹子少是因為台灣妹子普遍不像中國的妹子一樣願意出遠門 在中國的妹子很多都是從偏遠城市到沿海大城市 為了討生活，這樣長距離的搬遷對於中國妹子已經是見怪不怪習以為常 加上中國人口多，基數大，因此在海外的中國妹子遠比台灣妹子多 近幾年台灣護照到許多國家比中國護照還要來的方便 因此越來越多的台灣妹子願意出國打工了。 一般來說，單身女子，或是兩人結伴的妹子，很容易讓海關起疑 近幾年澳洲、紐西蘭、英國、加拿大、澳門、香港、新加玻、馬來西亞 都是很多妹子上班的聚集地，反而美國對於台灣妹子還是算比較少的 美國線來說，目前正在開發中 絕大部分是來美國上夜總會的，也就是台灣熟知的酒店 之前從某位藝人投資的塞班島、美國關島、紐約、洛杉磯等等區域 目前很多經紀都是在牽這些夜總會的線 一般價位都是保底$4000，素質越高保越高 台費午場(5:30-8:30) $30/ 小時，晚場(9:30-1:30) $60/小時 一桌最多兩個小時，午場內框外框都是約$300 晚場內框$300，外框$400。一般會有宿舍，會不會要付房租看公司 一般來說上班的時間，上超過兩個半月會補機票錢，補多少也是看公司 大部分都是要你一個禮拜上六天 而我在這邊奉勸各位苦命上班朋友，拿免簽證來美國，千萬不要待超過2個月 剛好2個月差不多，如果為了那一點公司補機票的小錢，而待到2個月半 下次要入境美國肯定被刁難，會不會被遣返這個都不好說 機票一張沒多少，不要拿入境機會開玩笑 而以上是夜總會的大概行情 至於像我們這種砲店會所，因為直接打砲，收入自然高很多 新聞裡的妹子帶了那麼多現金，實在令人匪夷所思，身上帶個1000-2000美金就夠了 在我們這邊都妹子都是煩惱著錢要怎麼帶回台灣，沒人會帶著大筆現金入境的 來算算吧 一天最少最少平均抓個$1000來算，等於是你上10天，最少就有1萬美金 很多條件很好的，一般一天平均衝到$2000不是什麼難事 新聞裡的案例來說，一週七天，如果是伴遊的話，那就是太少了 如果是做砲店的，那一樣很少，新聞說十多萬，我當她是19萬台幣好了 19萬台幣約6500美金，一週不到7000，算是免強達標 新聞裡有提到打炮是是10萬台幣的話，一週只有3500，那就非常浪費時間了 新聞有提到，妹子主動跑去找客人的這種模式 基本上非常少，因為沒有人牽線，這是不可能有門路的 已經出了國了，就是一定要賺到錢，不是賭博式的方式在操作 真的要抓賭場客，澳門就一堆中國大咖了，怎麼會跑去紐西蘭呢? 這種通常都是有客人找上當地的領班，而領班在跟台灣的經濟調妹子過來的模式 因此我之前有提到三個重點 1. 不要打扮得太漂亮 無法扮醜至少帶個大眼鏡，不要化妝，穿的樸素一點，不要犯賤 2. 手機裡的東西要注意 不管是對話、圖片、影音、甚至是貼圖。都不要引人遐想 最好的方法就是直接解除安裝，過海關在裝回來 3. 說謊前至少做點前置作業 你說要來美國玩，那只少知道要去哪玩，至少打一份行程 你說要來看你朋友親人，那至少把他的所在城市或者名字電話給記著 你說要來找男朋友，至少準備個幾張異性合照 拉你進移民局的小房間除了要嚇你講實話以外，另外就是做第二次確認 往往許多人都會被這招嚇到 美國因為惹了一堆人，因此很怕被打，所以海關對於入境的人審核特別的嚴格(不友善) 相對的程序就比較多，有很多事情都會打個備註在美國的人口資料庫裏面 一旦是被遣返過一次，可能就要有5-10年無法進入美國的心理打算 最後， 許多要外出賺點外快的妹子除了海關怎麼過要知道 老闆也要好好的選，歡迎各位來洛杉磯旅遊。 -------夢境結束線------- 以上純屬夢境，如有雷同，一定是巧合啦。 ※ 引述《jailkobe5566 (監獄狗鼻哥)》之銘言： : 1.媒體來源:TVBS : 2.完整新聞標題: 出國賣淫！2台女聲稱打工　遭紐海關攔下 : 3.完整新聞內文: : 說要出國玩，但手機裡面的照片卻好奇怪，遭到海關攔下拒絕入境！原來最近出現一種新 : 興模式，女子就是去國外援交，根據瞭解一週就能賺進10多萬，因為最近有真實案例，2 : 名台灣籍女子帶著大筆現金要去紐西蘭，但申請說要去打工遊學和實際上似乎有出入，海 : 關查驗手機，還發現好多奇怪的清涼照片，懷疑他們是入境賣淫，根據知情人士透露，這 : 樣的狀況真的越來越多。 : 入鏡紐西蘭，卻被海關在手機裡看到清涼照片還有大筆金額，2名台灣女子遭到海關攔阻 : ，被拒絕入境，因為紐西蘭懷疑這兩名台灣女子是要入境賣淫。 : 記者李香儀：「出國飛出去了做什麼沒有人知道，就有人把援交場景拉到國外去，一次賺 : 上數萬元，還可以出國玩，海關只能靠手機資料照片來攔阻一切。」 : 知情人士Peter：「不用說英文都可以上班，大部分都是在洛杉磯，你台灣價碼一樣收1萬 : ，在香港也收1萬，可是幣值不同。」 : 台灣青少年打工遊學逐年增加，但警方發現不少18歲到26歲的女生明著遊學，暗地賣淫， : 多半都到澳洲、紐西蘭等國，尤其以有賭場的國家為主，在裡頭尋找金主，再討論價碼。 : 曾辦相關案件警官薛博文：「每一個小姐去不一定賺到錢，有的可能沒被看上，供應妳吃 : 飯就回家，期間你可能要花自己的。」 : 以在澳洲來說，單純吃飯聊天1小時有2千台幣，賣淫1次有2萬，要是有知名，1個禮拜還 : 可以拿到10萬元。 : 曾辦相關案件警官薛博文：「萬一像去美國被查獲遣返，他們就是被遣返回來而已，遣返 : 回來唯一壞處就是，像美國比較嚴他們就會註記，註記妳就不能再入境，但是出了國就沒 : 辦法，這些畫面你不會有，我們頂多能用依據出國紀錄、帳單去做分析。」 : 儘管知情，但出了國就是很難抓，只是想不到單純出國竟暗藏春色，國外援交竟成了另一 : 種管道。 : 4.完整新聞連結 (或短網址): https://goo.gl/b5gE9n : 5.備註: -- 你問我答 https://www.facebook.com/LAGTOTheBusinessman -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 104.175.229.80 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1505378126.A.EC1.html
