: 本周一(20)，紐西蘭 11年級(15歲)的學生在參加由紐西蘭學歷評估委員會(New Zealand Qualifications Authority, NZQA)舉辦的國家教育成就證書(National Certificate of Educational Achievement, NCEA，編註) 第一級數學考試後，立刻出現了排山倒海的抱怨。 : 編註：這個認證一共分成 3級，其測驗結果可說代表了紐西蘭高中生的成績單。 : 明明學校沒有教 : 許多學生和老師在受訪時指出，試卷有許多題目題意不清、給的資訊不夠完整、甚至根本超出了 11年級的授課範圍。 : 像有人就指出題目出現了 12年級才會教的二次方程式，也有人指出試卷出現兩題幾乎一模一樣的題目，意味著只要學生不知道答案就會被扣上兩倍的分數。 : 也有學生認為，這次的考試難到沒有鑑別度，完全失去了測驗的初衷。 : 第二級考試比第一級容易 : 在接受採訪時，不只一名學生提到考題遠超過他們能夠回答的範圍，雖然他們一開始覺得可能是自己準備不足，但在考完後就發現其他同學也一樣困惑。一名 11年級的學生同時也有參加內部舉行的第二級數學考試(編註：給 12年級的測驗)，她也表示第一級數學考試比第二級還要難。 : 沒有鑑別力的考試 讓大家倍感壓力 : 11年級的梁凱利(Kelly Liang)則提到，她很擔心這樣的結果會影響到她後續申請大學的情況，她說：「有些人說這次的考試並不是設計給學生的、也跟 11年級的課程內容不相符......這一切只會讓充滿壓力的考試季更令人焦慮。」 : 「一場正確的試驗應該是要測驗學生有沒有吸收他們學習的東西，(我覺得)紐西蘭學歷評估委員會並沒有做到這點。」 : 其實老師也不樂見這樣的結果，因為他們擔心這會讓學生更害怕數學。 : 老師也苦惱的題目 : 許多老師也覺得這次的考試不符合 11年級的程度。奧塔哥省數學協會主席、洛根公園高中的數學老師弗雷澤(Amanda Fraser)便指出，就連她和其他數學老師也對部分試卷題目感到棘手，她也擔心這種打擊學生信心的考試只會讓學生更害怕數學。 : 擔心大家會更害怕數學 : 弗雷澤說：「對學生來說，數學彷彿有種無法跨越的屏障，我們已經很努力在打破這種汙名，而這(份試卷)絕對沒有幫上忙。」 : 經營網站「紐西蘭數學」、在Kapiti大學任教的數學教授威爾士(Jake Wills)也抱持類似看法，認為這次的考試既無法檢測出學生的能力，也完全是在扼殺學生學習數學的興趣。 : 其實這不是第一次紐西蘭學歷評估委員會接獲類似抱怨，但是該委員會強調考卷沒有問題。 : 22名老師連署公開信 : 目前為止，已經有 22名數學老師連署一封公開信給紐西蘭教育部和紐西蘭學歷評估委員會，表達對這次考題的不滿。 : 強調考卷沒有問題 : 對此，紐西蘭學歷評估委員會已經強調，第一級數學考題是「由一群專業的數學老師、根據正確的課綱範圍所設計」。 : 紐西蘭學歷評估委員會的副局長基爾凱利(Kristine Kilkelly)便表示，學生應該會發現有些題目比一般的還要難，但是這些題目是設計來測驗出程度更好的學生。 : 不是第一次 教育部長要查 :
