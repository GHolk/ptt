標題Re: [問卦] 為什麼不能偽造MAC位址?
時間Thu Jan 11 13:55:38 2018
大部分的 OS ( 以 windows XP以後來說 ) 都把這個功能鎖住。 以前小弟用 windows xp + hp mini 時在 windows xp 下就可以改 windows 7 之後好像全部都鎖起來。 後來在 Linux Ubuntu 下也可以改， 甚至只要支援的網路卡型號都可以。 因為有一些免費 ISP 是以 mac address 來訂 IP ( 請自行 google DHCP ) 如星X克與 ITaiwan 免費上網 一個 mac address 可以免費用 30 分鐘， 原則上改來改去就你用大家就不必用了， 還有改 mac 造成網路衝突和斷線問題。 不過很多平板或手機用免費上網抓片或 追劇那就是另一個問題了 現在大學的宿網也多會鎖 mac 及限流量。 還是別拿來做壞事比較好。 無線網路的 wep, wpa, wpa2 破壞破解就是另一個故事了。 別問，很可怕，不想被偵九隊盯上。 ※ 引述《Rusnake5566 (魯蛇+56 PTT最強帳號)》之銘言： : 大家都知道每張網卡都有製造商給他的獨一無二的MAC位址 : 全世界就只有他用這組號碼 : 難道沒有別的方法可以偽造一個嗎? : 有沒有不能偽造MAC的八卦? ----- Sent from JPTT on my InFocus M812. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 210.128.207.35 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1515650144.A.298.html
