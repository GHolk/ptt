※ 引述《Angelopoulos (失雨的小草)》之銘言： : 有障礙的 : 應該多少也有錢吧 : 去找阿公店弄一下 : 是很難嗎？ : 為什麼一定要人免費、自願幫你弄？ 很難。 首先是沒錢，就算有錢也都要支付各種照顧跟醫療支出。 或是，可以參考一下最近「新巨輪」相關的文章。 實際上，就連一天只能吃一餐的人，也都來申請過： https://www.handangel.org/?p=2620 「小折每週有六天共12小時來照顧他，也就是說每天有二小時， 居服員來幫他擦拭身體（就是乾式的洗澡）， 來幫他處理大小便，帶飯來給他吃。」 「一週中的第七天，他要自己自費僱請看護工。 一個完全沒有收入的障礙者要自己付錢，您沒有看錯!!」 其次是場地問題。 雖然推文中有人提到「先有合法性專區吧…到時自然店家會做無障礙環境」 但就算真有性專區了，應該也會被歸類在八大行業 －－請問大家有看過哪家 KTV 、三溫暖、舞廳、酒吧有落實無障礙嗎？ 既然現有的合法娛樂產業根本就不屑賺障礙者的錢，政府也沒有打算落實無障礙法規， 那麼就算有性專區了，中小型的性交易場所也不會有無障礙的啦。 （附帶一提，金錢豹那種大型的倒是有電梯， 可以讓不同身分的客人不會彼此遇到。 但我猜內部大概還是有很多門檻跟階梯吧） 關於沒有辦法出門的，雖然有人主張可叫外送。 但會到「沒有辦法獨自出門」程度的障礙者， 通常意味著常常有家人、看護、居家服務員或個人助理在旁邊。 而且很多障礙者的家人為了「方便照顧」，根本沒有設置房門， 在這種情形下，請問要怎麼在保有隱私的狀況下叫外送？ 就算有勇氣跟照顧者開口講這種需求， 請問照顧者會不會惹上「媒介性交易」的相關麻煩？ 而且，大家誤以為目前性交易法規的認知是「只要在專區內就行」， 但其實根本就還有規定必須在事先登記的營業場所才行。 請問就算有專區了，要叫性工作者到家裡來，還得要先把家裡登記成營業場所嗎？ 從這個角度來看，性專區制度根本只是延續違警罰法時期的政策， 完全沒有考慮到障礙者（包含老年失能）的需求， 而且還規定性工作者必須要「靠行」，所以對於剝削問題也沒有幫助。 （附帶一提，專區制根本不是2011年修法時的新制度，而是一直都有。 敬參拙作： https://plainlaw.me/2018/04/22/sex-transaction/ ） 到這邊應該會有人說那就到旅館吧。 但問題基本上是一樣的：沒法自己出門的重度障礙者， 要怎麼回應親友質問的「你沒事去旅館幹嘛」？誠實回答「嫖妓」嗎？ 另外，有哪家旅館會願意登記成性交易營業場所？ 更何況，目前多數旅館的無障礙房根本就不夠，或是只能推進房間但推不進浴室。 之前辦活動場勘的時候，遇到一間旅館還「很自豪」地說： 「我們有42間房，沒有無障礙房。」 我覺得吼，「手天使是不是假道學」不是很重要啦， 「道學」是甚麼？障礙者的處境是甚麼？現實困境是甚麼？ 討論這些才更重要吧？ -- ╭ ╭ ╭╮╭╮https://kong0107.github.io/ │ Υ ╭─╮ ╭─╮ ╪ ││││ ╮ ╮ ╭── Υ ╭─╮ │ │ ╞═╛ │ │ │ │╰╯│ │ │ ╰─╮ │ │ ╰─╯ ╧ └─╯ ╘═╛ ╯ ╯' '╰ ╰─╯ ──╯ ╧ └─╯ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 124.155.182.221 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1526723420.A.97B.html
