標題Re: [問卦] 微軟作業系統市佔怎麼那麼高呢？
時間Sun May 20 16:28:20 2018
傻瓜，不當然是信仰和愚民， 不然你以比爾蓋茲二十年前的世界首富， 是怎麼賺來的。 玩電腦夠久的人應該都聽過 PC 和 DOS， 但很少人知道在 DOS 時代微軟就幾乎獨霸， 也應該很少人知道 DOS 不是只有微軟一家。 那時文書還是很簡單的 PE2，中文還要掛 個倚天。 然後微軟抄來了 GUI 圖形界面系統，一說是 從迪吉多 "學習" 來的。 好笑的是，當年的蘋果麥金塔電腦比微軟更 早用，然後靠著 Windows + Office 麥金塔整 個被打趴。 那時小魯剛準備上大學，還買了一套正版 Windows 3.0。 接下來 Windows 幾乎從大學的基礎資訊科 學全面占據 ( Windows 3.1 要在 Dos 下啟 動，網路還沒有寬頻，還在用 28.8K 數據機 撥號 ) 資訊科學 = 微軟 Windows +MSOffice 基本上除了資工系的，大學老師也沒人也不 會教別的。 本來 1995 年還有一套 IBM 的 OS/2，應該大多數人聽都沒聽過。( 不過小弟也只裝過 一次，真的難度太高而且支援軟體太少) 然後差不多同時，有一套 Linux，大學裏除 了架站和學寫 C 語言程式，幾乎沒什麼人 在用。而且重點是，除了網路討論區，幾乎 沒什麼發問的地方。另一個重點， 學校教授也沒人會.... 想學得自立自強。 ( Linux 本來就是從一個芬蘭大學生開始發 展的，而且拜 Internet 之賜，發展成全世 界幾乎占有率前幾名的伺服器軟體 ) 不過當年交大出了張"雲觀"，應該是不少玩 Linux 的啟蒙光碟。 然後，教育部推展所謂 "資訊教育"，從師資 培訓開始，靠腰，全部被微軟綁標。幾乎從 教電腦的老師培訓也都只知道 MS Windows 和 MS Office。( 中間教育部曾經支持過一段 時間 Linux 的教育訓練，有個 EZGO 的光碟 。不過上完課幾乎沒人在用，連上課老師自 己都不會用了怎麼叫學生用。 本來政府想推另一套免費的 Office 系統， 結果全面和現行公文或文件格式不相容， 列印什麼的會跑掉，後來新的微軟 Office 檔案改成 docx 模式，更不相容，就越來越 少人用，後來也就不了了之。 現在的學校裏幾乎是由學校跟微軟買全校 授權，畢業生還可以畢業前買一套便宜的 MS Windows + MS Office，幾乎現在的 PC 或筆電也都出廠就綁死一套。其他的 作業系統除了蘋果以外沒人可以匹敵的。 因為全部都被綁死了。 另外就是玩單機 PC 遊戲或網路 PC遊戲， 除了微軟以外你沒有其他選擇。 好在手機因為微軟當年大意失荊州，不然 今天也未必是蘋果安卓二分天下。( 微軟 當年在 PDA 市場依然很強，幾乎把 PALM 打趴 )( 另外，Android 是從 Linux 發展出 來的，而 IOS 和 MacOS 是從 UNIX 發展 出來的，特別是 UNIX 是從1970 年代就出 現的，是比微軟還早的老前輩。 ※ 引述《ss0929 (999)》之銘言： : 去看販賣電腦地方 微軟還是佔大部分 : 而蘋果系統好像比較少 : 是否是市面上相容性微軟比較好呢？ : 有沒有這方面八卦呢？ ----- Sent from JPTT on my InFocus M812. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 126.70.2.164 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1526804905.A.948.html
