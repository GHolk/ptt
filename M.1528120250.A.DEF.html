標題[新聞] 微軟以 75 億美元收購 GitHub
時間Mon Jun 4 21:50:48 2018
1.媒體來源: INSIDE 2.完整新聞標題: 【快訊】微軟以 75 億美元收購 GitHub，看見三個未來的機會 3.完整新聞內文: 微軟正式宣佈，以 75 億美元收購 GitHub。GitHub 目前有超過 2800 萬名開發者， 在平台上持續共享和協作，以軟體創造未來。 兩家公司將共同推動開發者在整個開發生 命週期的每個階段可以實現更多目標，並且加速企業對 GitHub 的使用，也將微軟的開 發者工具和服務帶給新的受眾。 微軟執行長 Satya Nadella 說：「微軟是開發者第一的公司，有了 GitHub 的加入 ，我們將會加強對開發者自由、開放和創新的承諾， 我們充分意識到我們的社群責任， 並將盡我們最大的努力讓每位開發者都能夠建立、創造並解決全球最緊迫的挑戰。」 根據協議，GitHub 將保留其開發者優先的風格，並將獨立運作，為所有產業的開發者提 供開放的平台。 開發人員可以繼續為他們自己的計畫決定使用任何程式語言、工具和操 作系統，也能夠自行決定將程式碼部署到任何操作系統、任何雲端和任何設備。 微軟公司副總裁 Nat Friedman 是 Xamarin 的創辦人，也是開源社群的老手，將擔 任 GitHub 執行長。GitHub 現任執行長 Chris Wanstrath 則將成為微軟技術研 究員，向執行副總裁 Scott Guthrie 匯報工作，開展戰略性軟體計劃。 Chris Wanstrath 說：「我對 GitHub 和我們的社群在過去十年中取得的成就感到 非常自豪，我迫不及待地想要看看未來會發生什麼。未來的先體開發產業是光明的，我很 高興能夠與微軟聯手幫助實現這個目標。」 微軟執行長 Satya Nadella 也在官方部落格提到，微軟與 GitHub 看到了三個未來 的機會： 1. GitHub 將繼續授權開發者在開發生命週期的每個階段 -- 從構思到協作到部署到雲。 開發者可以自行決定如何進行開發、與誰合作，以及如何部署自己的程式碼。 2. 有了微軟的銷售和合作夥伴通路，以及微軟全球雲端基礎架構和服務的導入，微軟將加 快企業開發人員使用 GitHub。 3. 透過 GitHub，也能把微軟的開發者工具和服務帶給新的受眾。 而台灣的科技相關產業，應該相當關心，過去採用 GitHub 服務卻跟購買 Facebook 廣告一樣面臨必須要代為繳稅的窘境，如今 GitHub 被微軟收購之後，是否有機會也由 微軟開發票了呢？ 4.完整新聞連結 (或短網址): https://goo.gl/ZbYJKN -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 59.115.124.227 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1528120250.A.DEF.html
