: 【拒絕漢字霸權】 : 7/12日晚上，朱學恆PO了一篇文，嘲弄行政院新任發言人，Kolas Yotaka，堅持使用羅馬 : 拼音拼寫自己名字的主張。在留言中，朱本人和網友們，充分展現了對台灣史的無知，並 : 對羅馬字的使用者極盡諷刺與嘲笑。 : 身為羅馬字使用者，身為台南人，我有必要出來說幾句話，來反對這種令人作噁的「漢字 : 霸權」。 : 朱學恆與他的粉絲們，想必不了解，也不願意去了解，羅馬字系統，在台灣這塊土地，尤 : 其在我們台南，已經有接近四百年的歷史。荷蘭殖民者，為了傳教的方便，依照新港社（ : 今天新市一帶）流通的語言，以羅馬字拼音書寫聖經，以及教義問答。在殖民的後期，他 : 們也嘗試拼寫其他部落的語言，譬如虎尾壟一帶。在當時，識字率當然不會像我們今天這 : 麼高，然而，就新港社及台南地區而言，羅馬字算是普及，甚至各村莊都有在地的，能夠 : 讀寫的老師。 : 這些後來被稱為「新港文」的書寫系統，並未隨著荷蘭人離開而消失，而是在這塊土地上 : 持續了近兩百年，作為契約文字。心裡埋藏漢字霸權的人們看到這裡大概要開始冷笑了： : 都已經是清帝國統治，幹嘛不一律使用漢字？這個問題，只有對於文化弱勢毫無敏感，對 : 文化治理比清帝國更沒有意識的人才會不知道答案：因為這不是他們的語言啊！用漢字作 : 為契約文字，等於把詮釋權讓渡到能閱讀漢字的少數漢人身上，絕對加速平埔族群地權的 : 流失。這件事情，等於是在為叛亂打地基，連清帝國的統治者都不會想要。 : 在朱學恆的原文下面，有新港社的年輕人出來說話了，說他自小，爺爺奶奶就用羅馬拼音 : 教導他閱讀聖經。我不確定，他讀的羅馬字是什麼，但我必須指出，在台南，以及在台灣 : 各地，「閱讀羅馬字」都是某些人的經驗，以及認同。不只原住民族群，也包含台語族群 : ，以及客語族群。台灣話的羅馬字書寫，從一百多年前進到台灣（客語的部分，很遺憾我 : 並不清楚），並持續成為台語文字書寫的一個選擇。在長老教會系統中，被稱為「白話字 : 」（pe h-ōe-jī）的教會羅馬字，使用得非常普遍。回到台南，學習羅馬字之後，我 : 很驚訝的發現，原來能夠讀寫羅馬字的人大有人在。這一書寫系統，或許沒有漢字來得普 : 遍，然而一百多年來，卻仍然有一群人在使用著，用羅馬字書寫著屬於自己族群的故事。 : 現在的問題，不是「要寫漢字還是羅馬字」，而是憑什麼「以漢字為標準來評價羅馬字」 : 。當各種羅馬字的使用，在這塊島嶼上有這麼長的時間，這麼豐富的文化積累，乘載著這 : 麼多人的經驗，以及認同，為什麼這一文字應該被毫無道理的反對？ : 只因為你們無知？ : 文字，不僅僅是文字，而是一種豐富而複雜的載體，乘載著文化，記憶，經驗，認同，美 : 感與精神，看待世界的觀點。否定，嘲弄羅馬字，不僅僅是在否定一個文字系統，而是在 : 否定一群人，以及他們依據自身經驗與認同，為自己做出的人生選擇。民主政治的核心價 : 值，在於守護多元性，守護每一種看待世界的觀點，每一種不同的人生選擇。而朱學恆和 : 他的粉絲們在做的，無非是在說：你不配有和我一樣的權利。原因呢？只因為我不瞭解你 : ，也不屑瞭解。 : 在政治上，「人」是由權利構成的。為權利設下資格，就是為人設下資格。你不配有權利 : ，這句話的政治翻譯是：你不配為人。 : 「不屑」，「傲慢」，「嘲弄」。在這個社會裡，這些展演彷彿成為一個網紅的必要特質 : 。這些中二作為的背後，潛藏著一種對於「獨特性」的想像：獨特的基礎，是展現「他人 : 不如我」。要展現這一狀況，比起思考或議論，更重要的是讓他人出糗，或者累積自己的 : 人望，號招群眾，在網路世界讓他人難堪。正因為如此，這種「獨特」對於溝通完全是負 : 面的，這些愛賣弄獨特的網紅們，只能先故作姿態確保自己高人一等，才有膽量說出自己 : 毫無內容的空洞觀點。當雙方對等時，他的觀點只能失語，然後一再故弄玄虛。 : 比起獨特，思考與溝通更需要獨立。那是一種能不依賴權力，抽離卻又能關照差異者的品 : 性。 : 留言截圖： :
