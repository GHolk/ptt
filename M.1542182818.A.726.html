本魯十來年前從113機械系畢業 個人對這門課的一點印象 機械實作算是大學四年級生的畢業作品 上學期是機械實作(上) 主要是複習工廠實習的一些機械操作 後半段教授會公佈作品主題還有可以報帳的預算 讓 學生分組提交作品proposal 然後在大四下 也就是機械實作(下)就會依照proposal把指定的作品完成 不過相比其他科系每天熬夜趕作品 這基本上不用花太多心力就能過關的課程 雖然號稱機械系 但我們的精力90%都放在課本理論上 除了大二工廠實習摸過車床洗床等 加工設備之外 很多人真的沒多少實作能力 能否作出多麼高深的作品就可想而知了 當年我們這屆的主題是桌球發球機 一組三四個人 但大家最後完成的作品也沒好到哪裡去 我們這組的作品還能勉強發幾球 然後就卡彈了 印象中還有連球都發不出來的 但最後教授還是讓大家pass了 後來回想這門課只是讓學生瞭解一個產品是從圖紙 原理 預算評估 到預排設計時程 到動 手完成作品 瞭解產品設計流程等等 是一個未來工程師的基本能力 這門課印象中只有一學分 但卻是必修 而大四階段很多人要嘛生活重心已經不在學校 或 是準備考研究所 或是已經推甄上提早進實驗室的 這門課沒過就等於是要延畢 所以教授也不會太為難大家 當然那是十來年前的事 不知道現在會不會要求得比較嚴格 但是靠花錢買產品來交差已經 等同於侵權行為 等這些小朋友進社會就知道有事態有多嚴重了...... ※ 引述《minisung (255,137,0)》之銘言： : 1.媒體來源: : 蘋果 : 2.完整新聞標題: : 交大機械系實作課爆作弊　學生網拍買成品交差 : 3.完整新聞內文: : 做不出來用買的！交通大學機械工程學系大四必修的機械實作課程，遭學生踢爆竟有學 生 : 「作弊」。學生爆料，老師在實作課上，要大家分組自行組裝出一台四軸飛行器，沒想 到 : 一組「成績很好」的同學，竟直接訂購一台「全新」的四軸飛行器，被眼尖的同學在拍 賣 : 網站「抓包」。授課老師表示，由於念在同學是初犯，決定給予他們改過自新的機會， 但 : 作弊一事為講求公平，還是會反應在期末成績上。 : 學生爆料指出，機械系大四必修機械實作課程（二）中，授課老師在課堂上要大家以8 千 : 元的預算，由同學分組自行組裝設計出一台四軸飛行器，並規定只能買材料自己加工組 裝 : ，不能買任何現成的東西。沒想到期中檢驗時，其中一組「成績很好」的3名同學，竟 在 : 網拍一間模型店直接購買一台「全新」的四軸飛行器來交差。 : 「對認真努力花時間熬夜做四軸的同學非常不公平」，學生表示，有人在網拍上發現這 組 : 同學的拍賣交易紀錄，還有賣家示範影片為證，同學向老師告狀後，老師原本說會處理 ， : 但事發至今已經兩個禮拜仍沒有下文，懷疑老師因為這組同學成績好、刻意偏袒，憤而 向 : 《蘋果》爆料。 : 同時《靠北交大》也有自稱是機械系助教的人發文痛批，「要被當了找方法沒錯啊，可 以 : 找人幫忙、可以問學長啊！你們這群成績好的人所謂的解決方法就是作弊啊？」、「直 接 : 買飛機來demo就爽賺三學分，還仗著自己成績好教授不敢動你們欸！這種心態到底怎麼 上 : 交大的啊？就是有你們這種只顧成績，一點能力都沒有的人存在，人家才會說交大機械 是 : 個笑話。」也有學生痛罵：「美國或外國大學作弊或抄襲都是直接退學的。」 : 對於這起作弊風波，授課的老師則表示，由於學生們已坦承犯錯，念在學生是初犯，決 定 : 給予他們改過自新的機會。也因成績還沒打，這些學生的行為，將會「表現」在期末成 績 : 上。（突發中心黃羿馨／新竹報導） : 4.完整新聞連結 (或短網址): : https://tw.news.appledaily.com/life/realtime/20181114/1466085 : 5.備註: : 無 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 42.72.155.234 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1542182818.A.726.html
