標題Re: [問卦] win8是不是MS的黑歷史
時間Sat Jan 5 17:56:53 2019
※ 引述《yokann (紐約Lady M)》之銘言： : 從dos win3.1用到現在win7 : 期間有買過win8筆電 (R930) : 幹 超級難用的 : 害我現在還是死守win7 連10都不願意用 : win8是不是微軟的黑歷史? : 有比vista還廢嗎? 趁有空時打打廢文 分析一下常被拿來鞭的微軟3套OS問題點： 1.Windows Vista 説Vista最廢，他只能排第三，但也不能說他完全沒問題，Vista會失敗，常被拿來鞭的有: 1.太低估硬體需求 雖然2005年當時的微軟跟你說只要512MB、800Mhz的CPU就可以跑了，你的舊電腦都可以跑 ，但是他沒跟你說的是，如果你要跑「非常順」，你的系統硬體可是要更高的。 這是Vista還沒推出前當時一般使用者常見的電腦規格： 1.CPU:Intel P4噴火龍 / PD噴火龍2代 / Celeron / Celeron D AMD就Athlon 64 / Sempron 2.RAM:大多落在於512MB的比較多，1G那時是主流 3.VGA:內顯居多的不少，當然也有裝低階顯卡的 4.HDD:IDE或SATA都可以 而實際上，Vista要完全發揮出他的本能(就是人家說的不頓很順)，則是要： 1.CPU:Intel Core 2 Due / Core 2 Qurd AMD Athlon 64 X2 2.RAM:32位元Vista建議2G，64位元則是4G 3.VGA:NVIDIA建議Geforce 6系列以上 ATI就是X1000系列以上 Intel G965/G945 Q965之後晶片組 4.HDD:要用SATA硬碟 至於像SiS VIA Intel 945以下這類的內顯因為硬體所配規格差，導致無法呈現Aero特效， 只能QQ。 此外，驅動程式也是另一個問題，由於OS核心架構大改(NT 5.x->NT 6.0)，一些原本在XP 或2000的驅動到了Vista卻罷工了，當時另一個現象就是Vista都推出了，結果廠商原本推 出的新機器卻沒有Vista用的驅動，結果裝置動不了，使用者就怒了。 2.軟體功能設計上的問題 拿一個最常被鞭的：使用者帳戶控制(UAC)，每當要調整一個系統設定，就會跳出來問你要 不要執行，不是說這個功能不好，但一些很基本的功能，不會影響到系統的設定也要跳出 來詢問。舊程式就慘了，當初都是以Administrator下去設計的，遇到Vista都必須按右鍵 以最高權限執行，遇到不懂的如何提升權限無法執行程式的使用者馬上就直接開罵了。 微軟想學Linux的sudo結果學的四不像，當時還蠻頗呵的。 雖然有些詭異現象在推出SP1時大多解決掉了，不過當年裝一個Service Pack要1小時，也 算經典了，而這樣的情況直到後面因為Vista不如預期、記憶體暴跌，一堆人買來直上8G， Vista被嫌慢的聲音也就漸漸變小聲了。加上隨後Windows 7的推出，此時大多數PC用戶的 硬體早以符合Vista的需求，Win 7順利無縫接軌，Vista也就漸漸的消失在人們的生活中。 2.Windows 8 要說最爛第二名，那就非Windows 8莫屬，Windows 8在硬體支援度來說，其實問題不大， 因為就NT核心來說變動幅度算小(NT 6.1 -> NT 6.2)，所以基本上能在Vista跟7跑的，8大 多也能跑得動，主機的規格亦也是如此。既然硬體不是問題，那最大的問題就只剩下軟體 了。 為了參與平板大戰，微軟決定把他的Windows作業系統大改造一番，把經典的「開始」功能 表大改，開始圖示拿掉，加入Metro App商店功能，還推出了一個詭異的口訣，教使用者如 何使用Windows 8。 不過微軟似乎忘記用鍵盤滑鼠的使用者居多，推出之後毫無意外的跟Vista一樣被酸的體無 完膚，雖然之後推出了Windows 8.1，有買Windows 8的都可以免費升級成8.1，但也成就了 Windows 7的地位更加鞏固。 Windows 8唯一作出的貢獻，就是可以用439元的價格買到正版Windows 8 OS，還可以免費 升級成8.1。 看到這裡，你是不是發現到了一件事？沒錯，這兩套系統，至少沒有「狂當機」 3.Windows ME 相對於Vista的硬體需求災難與Windows 8的使用者介面災難，Windows ME對於微軟來說， 可以說是心中永遠的痛，因為前兩個OS，至少不會三不五時藍色畫面當機。 在微軟推出Windows 2000之後，沒多久就推出面向家用版本的Windows ME。事實上，當時 在開發Windows 2000的時候就有謠傳說要開發「基於NT核心」的「Windows 2000 家用版」 ，但後來喊卡後就變成了現在大家看到的：拿9x的核心改了一下變成了Windows ME。 優點就是把以前在95 Plus跟98 Plus推出過的彈珠台跟連環新接龍整合進來，再加入5款 Internet小遊戲，多了網路安裝精靈、Windows Media Player、Movie Maker、自動更新 還有「系統還原」。 缺點就是因為拿掉了9x的特性:簡單來說就是拿掉了MS-DOS部分功能，導致原本以32/16位 元混合的OS比起同類型核心的98還要更不穩，執行程式效能會有變慢的現象。 另外就是「系統還原」這個功能，在當時算最狂噱頭，但是設計上明顯瑕疵，還原了結果 原先裝的程式開不起來，有的一還原就直接死給你看。到頭來系統還原只能默默的關掉， 當作沒這個功能。 結論： 綜合以上三個微軟作業系統，沒用過Windows ME，你就別說Vista跟8是最爛的。 題外話：我很恨我當時的兩位國中同學為什麼要買裝Windows ME的電腦，三不五時就跑來 問跟我求救，x的，不知道當時多少工具人恨死這套系統了？ -- 咕嚕靈波（●′∀‵）ノ♥ 歡迎加入真步真步王國的行列 https://i.imgur.com/00tjY2j.jpg
