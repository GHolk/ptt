標題Re: [問卦] 鼻子過敏是不是只剩移民一途?
時間Tue Mar 19 21:12:04 2019
這個似乎是台灣的氣候導致的問題 1.濕度高讓塵蟎易於滋生 2.溫度高塵蟎活性增加 而日韓美有兩個優點 1.濕度低所以降低塵蟎滋生 2.氣溫低塵蟎活性也降低 另外過敏還有幾個影響因素 化學性：各種過敏原以及飲食，ex：塵蟎、海鮮、蛋、花生等 物理性：氣候以及環境溫度、濕度變化等 因此要改善過敏，可能幾個方式 a.環境控制 i.降低濕度並改善空氣品質，使用更好除濕機以及空氣清淨機 ii.減少環境布料成分 iii.使用抗塵蟎的寢具 b.透過藥物控制，ex：抗組織胺、鼻噴劑 c.飲食注意 d.降低周遭的環境變因 至於開刀是否好就見人見智了 因為有的說有改善有的不一定 只是因為可能的根本原因是氣候，在台灣就比較難解了 不知大家覺得呢 ※ 引述《cattgirl (聖女貞德)》之銘言： : 特別回一篇yoyo老農 : 我也常常出國，日本韓國空氣都是乾冷，反觀台灣又濕又潮是濕冷。 : 台灣鼻病問題特別嚴重，建議廢文心得文大師到臺大醫院耳鼻喉科檢查， : 有個疑問，這種疾病年青時就該開刀治療，為何yoyo大叔硬是拖延到70歲才開始該該叫? : 住院治療大概兩周 : ※ 引述《yoyodiy (廢文心得文大師)》之銘言： : : 林北鼻子嚴重過敏 永遠只有一孔會通 有時甚至兩邊不通 : : 很難過 : : 上禮拜去了韓國 神奇的是一下飛機沒多久鼻子就暢通了 : : 在韓國期間也不會流鼻涕 打噴嚏 好像就沒過敏這個問題 : : 可是回來沒多久 噴涕鼻塞又來了 : : 整個潮濕的環境讓鼻子過敏又自動復發 : : 要徹底解決鼻子過敏是不是只能移民了? : : 八卦否? ----- Sent from JPTT on my Sony G8142. -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 180.177.1.161 ※ 文章網址: https://www.ptt.cc/bbs/Gossiping/M.1553001132.A.F12.html
