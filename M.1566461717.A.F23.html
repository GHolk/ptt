標題Re: [新聞] 傳三星良率出問題 高通5G處理器全報廢
時間Thu Aug 22 16:15:15 2019
有感而發來聊一些陳年往事， 先問問鄉民自己認為高通是一家什麼樣的公司？ 半導體公司？IC設計公司？晶片公司？通訊公司？ 答案是都不對， 高通上層對自己的認定是一家以專利為中心的法律公司， 這也就可以解釋高通一連串在代工廠以及晶片團隊的怪異處理方式， 高通曾經投資非常多的錢在桃園成立高通光電研發Mirasol顯示器， 內部員工在All Hangs裡頭， 直接問CEO為何要發展跟本業差這麼多的東西， CEO回答他們並沒有要量產拿下整個顯示器市場， 高通主要的目的是拿下顯示器大部分的專利， 讓以後所有的顯示器公司都必須繳專利金， 但是後來失敗了 同樣道理可以套在LTE, 4G, 5G上面， 高通最主要目的就是拿下專利， 並且有proto-type以及少量但是最早在市場的產品， 這樣對他們在專利訴訟上有非常大的優勢， 至於工程團隊，晶片代工廠等事情，就不是公司最在意的事， 高通在代工廠出包已經非常多次了， 我想到在2012年的時候， MSM8960 Snapdragon 大賣，賣到完全缺貨， 我問我前輩缺貨可以馬上跟台積電增加訂單呀！ 前輩說我太菜，追加訂單都要排在半年甚至一年之後， 現在加訂單根本來不及解決缺貨問題， 我說那為何當初在下訂單的時候沒有下多一點，或是分批加單， 答案是"高層根本沒想過會大賣到缺貨"， 最後解決方法是工程團隊人事凍結，一年不加薪沒分紅， 高層Marketing/Strategy 拿下大功勞因為產品大賣又沒有任何浪費的晶片產出， 專利部分也成功拿下4G LTE, 類似情況在2014年的Snapdragon 810又發生， 當時64位元的Kryo已經研發完成，內部驗證全部通過， 但是高層先以64位元市場未成熟為理由把Kryo先壓下來， 接著突然發現錯過64位元市場， 聽信Marketing/Strategy高層的鬼話採用Cortex-A57， 結果產品過熱問題一堆， 但是當初內部驗證810的時候某個高層一直說all pass, 最後解決方法是CPU team大裁員，最後全部解散， 把這個錯誤全部推給CPU team， 那個堅持用Cortex-A57說all pass的人發大財升大官， 因為他拯救了錯過的64位元市場 所以這次三星的事件也不用太意外， 因為上層從來沒把高通當作晶片公司在看待， 他們只在乎專利 ※ 引述《MediaTekInc (發哥)》之銘言： : 傳三星良率出問題 代工高通5G處理器產品全報廢 : 〔財經頻道／綜合報導〕有市場消息傳出，南韓三星（Samsung）在代工行動處理器大廠 : 高通 （Qualcomm） 的中階5G處理器時發生良率問題，導致該產品全數報廢的消息，引發 : 外界關注。 : 根據《財經新報》報導，引用知情人士消息，指出三星在代工高通中階5G 處理器 : Snapdragon SDM7250時，良率出問題，導致全數產品慘遭報廢。且不僅是高通 : Snapdragon SDM7250處理器良率出問題，也傳出三星本身的處理器也同樣發生相同狀況。 : 不過據報導初步盤點，此事件對5G手機需求市場的影響並不太大。 : 知情人士稱，由於高通Snapdragon SDM7250處理器預計將在明年第1季問世，因此對三星 : 來說，還有一些補救的時間。且對三星來說，高通這顆處理器對為目前該公司最重要的客 : 戶產品，因此三星勢必會盡快解決良率問題。知情人士也指出，倘若三星屆時不幸良率還 : 是偏低，預估仍還是可以達到少量出貨的水準，不會陷入完全無法供貨的窘境。 : 外界也好奇，三星大出包，是否會影響與高通後續的合作關係？知情人士認為，倘若後續 : 高通Snapdragon SDM7250處理器因良率問題出貨變少，將會影響三星在7奈米製程需求上 : 的佔比，同時也將削減客戶對該公司7奈米製程的信心。不過知情人士也指出，解決良率 : 問題的能力，三星也非等閒之輩，因此之後是否會有持續性問題產生，有待後續進一步觀 : 察。 : 另外，報導也指出，倘若未來三星良率問題，影響到高通在5G處理器上的供貨狀況，將有 : 助於聯發科進一步搶食市場，提升該公司5G處理器的市佔率。 : 來源：https://ec.ltn.com.tw/article/breakingnews/2891135 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 73.222.35.14 (美國) ※ 文章網址: https://www.ptt.cc/bbs/Tech_Job/M.1566461717.A.F23.html
