分享個人對國土重劃的一些想法。 首先，台灣目前的行政區太複雜，例如光是「市」就有直轄市、省轄市、 縣轄市之分，而且大家都想拚升格，覺得"市"比"縣"高級。 另外，應完全廢掉「省」的概念，並將目前的縣市級行政區都改成直轄的 一級行政區，並且改稱「州」。 這裡的州指的並不是美國的州(state) ，而是prefecture，也就是日本的 都道府縣的概念，日本的都道府縣的英文也都是prefecture，有鑑於大家 都要求公平，所以一律使用"州"，其實也不一定要用"州"這個名稱，總之 就是都是一級行政區，地位皆平等，不再有目前縣不如市然後一直吵升格 的情況。 因此各級行政區簡化為：國家(state)─州(prefecture)─區(district)， 區長由州的行政長官指派。 道路也需要調整，除了高速公路(Freeway)、快速道路(Expressway)不變， 省道(Provincial Highway)→台X線(State Highway)； 縣/市道(County Highway) →州道(Prefectural Highway)； 鄉/區道(Country Road) →區道(District Road)。 接下來是行政區劃分，以下只討論州層級，至於更細微的區之重劃則暫不 討論。我的劃分其實和目前的行政劃分差別不大，因為許多縣市都不想因 為被"合併"而喪失主體性，也考量台灣的人口數、生活圈、合併複雜度等 問題，故沒有太多變化。 1.基隆州：原基隆市+原新北市之汐止、萬里、金山、石門、瑞芳、平溪、雙溪、貢寮 2.新北州：原新北市，扣掉劃分給基隆州之區 3.台北州：原臺北市 4.桃園州：原桃園市 5.新竹州：原新竹縣+原新竹市 6.苗栗州：原苗栗縣 7.台中州：原臺中市 8.南投州：原南投縣 9.彰化州：原彰化縣 10.雲林州：原雲林縣 11.嘉義州：原嘉義縣+原嘉義市 (嘉嘉雲也可考慮合併成雲嘉州) 12.台南州：原臺南市 13.高雄州：原高雄市 14.屏東州：原屏東縣 15.台東州：原臺東縣 16.花蓮州：原花蓮縣 17.宜蘭州：原宜蘭縣 18.澎湖州：原澎湖縣 19.金門特別縣：原金門縣 20.馬祖特別縣：原連江縣 (有點國際法概念的人應該會知道金馬和台澎其實不一樣，涉及二戰後 的戰敗國領土代管問題) 歡迎大家指教。 ※ 引述《turbomons (Τ/taʊ/)》之銘言： : 公視新聞 : 柯拋基北合併議題 基市長林右昌提重劃 : 林孝儒 彭耀祖 : 基隆報導 : 台北市長柯文哲拋出基隆市併入台北市的構想，今天基隆市長林右昌表示，很樂意在選後 : ，與柯市長及新北市長侯友宜共同討論。侯友宜則對於北北基要合併，樂觀其成。 : 台北市長柯文哲再拋國土規劃議題，點名基隆若要有最大發展，就是併入台北市，還批評 : 雙北市包在一起，卻都是直轄市，太荒謬，力促「首都生活圈」的願景。 : 柯文哲表示，「正確的講法叫首都生活圈，也不是說一定要在行政上併進來，其實應該基 : 隆到楊梅這一帶，大概也有一千萬（人口）。」 : 柯文哲再推國土重劃，基隆市長林右昌表示，樂意在選後與柯市長及新北市長侯友宜，共 : 同討論行政區劃問題，同時也針對台北市發展停滯，認為是缺乏振奮人心的發展計畫，和 : 腹地嚴重不足，其實台北更需要基隆。 : 林右昌指出，「其實是台北市更需要基隆港市，包括整個北海岸、東北角、新北各區，以 : 及包括基隆港市等等，未來都應該整體來思考，是否成為首都的一份子。」 : 「市長支持北北基合併成一個首都嗎？」對於記者提問，林右昌回應，「我覺得這樣的談 : 法太過簡化。」 : 林右昌表示，合併可以有多種方案，包括重新規劃後，一加一加一除以三，或是以淡水河 : 為界，一加一加一除以二，都可以是選項。新北市長侯友宜則指出，國際大城市本來就需 : 要上千萬的人口紅利，對於北北基的合併樂觀其成。 : 侯友宜表示，「北北基一個大生活圈，本來就是要共同合作，共同努力，所以三個（城市 : ）要合併，其實我個人是樂觀其成。」 : 至於民進黨立委蔡適應認為，基隆是否併入北市，應由基隆市民自行決定，基隆市議員宋 : 瑋莉則建議用公投，來反映市民意見。 : https://news.pts.org.tw/article/458432 : https://youtu.be/j2kbfG3LoXY
