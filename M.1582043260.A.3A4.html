另外 APP 本身在啟動時會偵測系統有無安裝指定的APP， 包含：淘寶、微信、抖音、美圖秀秀還有 Google Drive(沒有錯是 GDrive)， 因為拆包只能拆到 API 等級，所以不知道偵測之後有甚麼作為就是了。 https://i.imgur.com/Kwylkvb.png
notifyViolateVoice 會在違規時發出很機掰的聲音，比如關閉煩人的通知 https://i.imgur.com/b2yTSvk.png
https://i.imgur.com/k1uXSQs.png
這個 LSApplicationQueriesSchemes 本來用途是用來串連其他應用， 可以分享SDK甚麼的，不過國軍APP有甚麼好分享給微信的XD，所以...... https://www.jianshu.com/p/a1895c766400 另外 APP 本身也會記載解鎖上鎖的時間點但不會上傳，資訊官有需要可以隨時調閱。 描述檔則是一個是應該是為了接入內網使用的(192.168.xx.xx)， 另一個則是鎖住相機的取用限制描述檔， 整個方案都是用蘋果自帶的解決方案。 另一方面，Android 則是由中"科"院自行研發的 APP，包含上鎖都是自研的， 安裝後離營需要解除安裝要輸入主官密碼才能解除，否則是刪不掉的。 (相較之下 iOS 則是解鎖之後就能自己刪掉 APP也能刪掉描述檔) 加上因為 Android 客製化機型多樣，對於不相容的情況也就更為常見。 : 2.建議手機資料全部先備份 : 資訊人員跟我們說 : 如果mdm灌不進去會強制重置或硬灌 : 灌到後來聽說有些手機會有小問題或死機 基本上帶進去不會強迫你重置，頂多就不給你用而已， 不過因為人數眾多，所以如果當下沒辦法安裝，要等下一次可能要等很長一段時間。 雖然是這麼說，但是因為安裝的當下會是無密碼狀態， 所以建議有私人重要訊息或者資料還是轉移到其他備用機上面， 周末放假在拿出來用會放心很多。 : 3.中國手機不能帶 : 就是這樣 華為、小米、Vivo、OPPO 等等手機都禁止在營內使用， 入營的當下會簽切結書，至於有那些是中國手機廠大家應該比我專業XD : 4.建議別帶剛新買的手機 : 因為有人的手機螢幕在軍營裡裂開了.. : 不確定是不是放手機時別人丟下撞到就裂開 : 還有一例是 : 有人推手機櫃去中山室時 : 手滑翻倒..…嗯 : 還有手機如果灌壞惹國軍不太會負責 手機因為數量眾多所以存放方式也比較粗糙， 有些有獨立手機櫃，有些則是有保鮮盒，還有一種用洗衣袋， 建議帶進去就有弄壞的最壞打算，反正用的時間也不長，還要裝怪怪的軟體， 雖然很多人都帶當年旗艦手機就是了， Zenfone 6 iPhone XR S9 .... : 5.mdm不要手滑刪掉 : 刪掉會..…飛起來，陸軍變空軍 應該比較不會有天兵刪掉，比較需要注意的是 iOS 如果為了繞開 200MB 限制去改日期， 改太多會造成企業級 APP(國軍 MDM)的簽證到期(簽完當下有效期 365天)， 一旦到期就會出現點開就閃退的情況，這樣就要重灌惹。 : 6.sony聽說不好灌mdm : 所以如果想使用網路，建議帶別隻。 Sony 怪問題特別多，我當時候是 Android Pie 手機因為版本太新不給裝， 同寢的幾天前剛買就被打槍，氣到直接買 iPhone 6 二手的XD : 7.查詢該營區訊號最佳的電信公司 : 有些營區就算是中華也無訊號 : 所以如果想使用 : 可以先查一下， : 再辦個該電信公司預付卡含網路那種 中華應該最好，台哥大好像很慘，台星我用起來是沒啥問題很奇怪XD 基本上看運氣啦，反正都不會滿格就是了 : 大概是這樣 大概是這樣，還好我退了 : ----- : Sent from JPTT on my Asus ASUS_X01BDA.
