https://www.dosbox.com/wiki/Cuesheet 重點就在這篇 DOSBOX 的 Wiki 說明了 DOSBOX 上如何掛載 Mixed-Mode CD 映像檔 也就是多半採用的 cue/bin 的映像檔 若有原始光碟, 在 Linux 上就可以透過提供的指令轉出 cue/bin 下半的 "Compressed audio tracks" 就是如何修改 .cue 檔的方法 1. 透過原始光碟或是 cue/bin 把音軌擷取出 .wav 檔案 2. Linux 下使用 oggenc *.wav 即可壓縮檔案 oggenc 還可以加上 -q N, N 介於 -1~10, 控制音質, 預設為 3 這裡也可以用喜歡的工具轉為 mp3 檔案 3. 修改 .cue 檔 * 對於每個 Track 開頭增加 FILE "OGG_AUD_FILE" OGG 這裡 OGG_AUD_FILE 是該軌對應的 OGG 相對路徑檔案名稱 事實上 MP3 也可以, 結尾 TAG 則改為 MP3 * 對於 INDEX 時間一律改為 INDEX 01 00:00:00 基本上就這樣, 比較可惜的是 VM 工具 (qemu, vbox, pcem) 不支援這模式 因此只能套用在 dosbox 可及的 dos / win3.1 遊戲 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 123.110.208.161 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/Old-Games/M.1628084128.A.655.html
