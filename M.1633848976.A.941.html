# 關於我自己 做過三份全職程式設計工作， 年資約八年，今年35歲 國立大學資工系學士畢業。 最後一份工作在關貿網路寫程式，政府專案組。 主要技能是微軟方案，.NET / C# / MVC /Web form 。 熟悉源碼掃描後改善與效能改善。 離職前月薪約42k ------------------------------------------------- # 我的面試準備： 1. 全彩履歷表包含中文自傳，五份。 2. DEMO自己離職後開發的框架（基於ASP.MVC 自己開發的架站框架）。 3. DEMO自己設計的框架導入教材。 4. 擬定上一份工作離職理由說帖，每次面試都會被問到。 5. 到leetcode，解了約10題easy的。 -------------------------------------------------- # 投遞履歷範圍： 高雄火車站以南，前鎮區以北， 薪資寫面議四萬以上的。 面試時間為2021/9月中～10月初 -------------------------------------------------- # 預期薪資： 一般開50K，期望拿到45K以上。 ------------------------------------- 下面的順序 ------------------------------------- #中山大學 圖書館資訊處 面試中得知這份工作是計畫人員， 我記得投遞履歷之後超過兩週才收到面試通知 其實很久很久..... 現場提供的薪酬參考表(一級)是30K， 根據我在學校待過多年的經驗，通常一年過了才會往上升一級。 而且年終只會有1.5個月 工作內容是：要獨立開發圖書館資訊系統的外掛(包山包海)， 用人單位是圖書館組那邊。資訊系統的部分我猜沒買API， 工作上我猜要寫parser去解析與做一些自動化的動作。 不過筆試沒有考正規表示法或多執行緒之類的。 ## 筆試 PHP 1. Post get 差異 2. Include require 差異 3. 99乘法 4. 費氏數列 5. Sql新增修改刪除 6. 程式執行結果 7. Cookie跟session差異 ## 加碼題(十分鐘) 全英文題目 → SQL選取比較特別的資料，看起來像是課本上的範例 ## 面試 面試官有四人，一位處長，兩位組長， 一位用人單位圖書館組的。 先自我介紹 然後面試官大概問了下列問題： 1. 英文能力 2. 薪資可接受範圍 3. 熟不熟資訊安全 結果： 通知另一位北科資工碩士畢業的畢業者去上班， 面試前跟他聊天時有留下他的聯絡方式 問了一下他收到的Offer內容 他薪資36K正式37K，看起來薪酬實在是太低了。 --------------------------------------------- # 元赫科技股份有限公司 電話面試，電話中表明要找高雄的主管職， 可是開的是工程師的缺（可能是找工程師薪資的主管職）。 電話面試官也是工程師，主管職的一些問題他不是很清楚。 要去台北二面，錄取後要去台北受訓兩個月。 但公司沒有提供臨時宿舍，需自己找房。 （不知道能不能住飯店） 房租似乎有補助，但金額不確定。 詢問加班文化，電話中表示為「責任制」。 （薪資部分也是到責任制的水平嗎？） 工程師有問一些簡單的技術問題， 像是效能改善有哪些作法。 Reflection用在什麼時候（映射出物件屬性） 但我因為找的是工程師職，加上要台北受訓兩個月， 就電話中直接拒絕了。 想找主管職跟台北受訓部分建議在職缺中先寫清楚 ------------------------------------------------- # 勝強科技有限公司 面試官有兩人，一位看起來是工程師， 另一位是人資兼客服。 辦公室小小的， 印象中大家都是是配一台14寸左右的筆電工作。 整個辦公室目測約八個人左右。 組成大致上是 工程師 + 一位業務 +一位人資兼客服。 工程師要負責需求訪談等等 工程師組成幾乎都是應屆畢業生。 希望有即戰力可以帶領開發。 目前開發以web form為主。 期望薪資，開55K。 尚未收到回覆 ---------------------------------------------------- # 集志科技股份有限公司 也是接案公司， 公司內職位只有工程師（兼任PM 客服 總之要負責專案所有事情）。 開發上好像也是配一台筆電而已 公司業務來源是承接二包或三包的部分， 由其他公司轉包專案出來，所以沒有業務， 本身也沒在選擇案子。 開發部分有點像是在架設CMS，CMS本身是自己開發的平台。 很強調一定要用這套框架開發 不過我現場看了程式碼 為了開發速度與彈性不顧資安與效能， 框架本身設計上有嚴重的injection問題 不過主管說他們不接政府專案不會被資安稽核， 所以沒關係。 我自己覺得這樣似乎缺少職業道德。 大部分時間在聽主管講自己的過去中度過了。 期望薪資，我開50K 尚未得到回覆。 -------------------------------------------------------- # 威芯科技有限公司 雖然寫新創，不過其實是做博弈的公司 面試官兩位技術，技術面試完換人資。 公司技術主軸是PHP（LAMP） 但要找.NET的人， 因為從中國買了另一款遊戲要整合進自家的遊戲大廳。 那款遊戲是C# Unity開發的 薪資我開55K， 面試官當面回覆錄取試用期給50K， 後續依照表現調薪 上班時間是9點-18點 薪資保障14個月 想協調是否能能八點上班到五點， 好讓我接送小孩 據人資田小姐表示： 每天下午五點都會全部一起開檢討會議， 及協調事項。 一定會晚於這個時間下班。 目測辦公室約20人，開發應該是筆電＋外接螢幕。 結果：錄取， 但每天超過一小時的檢討會的文化讓我害怕，因此婉拒。 ------------------------------------------------- # 派維爾科技 公司主要做多元金流的服務，與銀行合作。 地點外觀有點像是民宅，不過有電梯。 內部擺設上很像傳產的辦公室， 伴隨著老闆的眾多玉石佛像收藏品。 開發看起來是桌機雙螢幕。 面試官一人，是主管 問了很多細的技術問題， 不過大部分時間像是在跟他聊天， 聊天起來感覺人還蠻明理的。 可以配合我接小孩時間讓我8-17上班。 我開50K，對方開試用期45K 調薪48K， 沒有保障14個月。 結果：當天通知錄取，回覆會去這間。 ----------------------------------------- # 校園團契 這是知名基督教福音機構， 面談前有表明可遠距WFH， 但每週二要去台北參加禱告會 （還是一個月一次有點忘了）。 整個開發團隊約四人， 但感覺分工還蠻隨性的，有點混亂。 薪資部分，到頂是40K， 年終一個月多，不會超過兩個月。 薪資差不多是我最底的底線。 開發上鼓勵用自己的電腦 （但似乎沒有補助，我建議每月至少補助個1~2K） 上班時間雖然好像蠻彈性的， 不過不確定能不能八點上工，最早要八點半。 結果： 原本以為是無聲卡，後來大過有回信沒上， 大致上是他們想找php iOS即戰力（我走.NET為主)， 雖然可以遠距，但遠距前要先在到台北資訊處磨合一年。 ----------------- 謝謝各位版友看到這邊。 -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 114.33.146.32 (臺灣) ※ 文章網址: https://www.ptt.cc/bbs/Soft_Job/M.1633848976.A.941.html ※ 編輯: nwish (114.33.146.32 臺灣), 10/10/2021 14:57:40
